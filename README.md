# Gesture Controlled Camera

A Python OpenCV project that controls camera zoom using hand gestures.

## Features
- Hand Tracking
- Gesture Detection
- Zoom Control
- Real-time Processing

## Technologies Used
- Python
- OpenCV
- MediaPipe
- CustomTkinter

## Run
```bash
pip install -r requirements.txt
python main.py
